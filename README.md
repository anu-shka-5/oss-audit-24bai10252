## 📌 Project Description
This project is part of the OSS Capstone where we analyze an open-source software (Python) and explore its Linux footprint, ecosystem, and ethical implications.

We also developed 5 shell scripts to interact with the Linux system and analyze software behavior.

---

## 📜 Scripts Overview

### 🔹 script1.sh — System Identity Report
Displays:
- Kernel version
- Username
- Uptime
- Linux distribution
- Current date

---

### 🔹 script2.sh — Package Inspector
Checks:
- Whether Python is installed
- Displays package details

---

### 🔹 script3.sh — Disk & Permission Auditor
Analyzes:
- Important system directories
- Permissions
- Disk usage

---

### 🔹 script4.sh — Log Analyzer
- Reads system logs
- Finds occurrences of keywords (like "error")

Run example:./script4.sh /var/log/syslog error

---

### 🔹 script5.sh — Manifesto Generator
- Takes user input
- Generates a text file expressing open-source philosophy

---

## ⚙️ How to Run

1. Navigate to project folder:
cd oss-audit-24bai10252

2. Give execution permission:
chmod +x script1.sh script2.sh script3.sh script4.sh script5.sh

3. Run scripts:

./script1.sh
./script2.sh
./script3.sh
./script4.sh /var/log/syslog error
./script5.sh
